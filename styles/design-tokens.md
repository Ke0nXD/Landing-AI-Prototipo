# AI Landing Lab Design Tokens

These tokens mirror a Figma-ready system for the current implementation.

## Color

- Background: `#050914`
- Surface: `#0b1220`
- Surface soft: `#101827`
- Border: `rgba(255, 255, 255, 0.1)`
- Text primary: `#f8fafc`
- Text muted: `#94a3b8`
- Cyan accent: `#67e8f9`
- Violet accent: `#a78bfa`
- Emerald accent: `#6ee7b7`
- Warning accent: `#fcd34d`

## Visual Tones

- Premium dark: dark base, cyan/violet accents, glass panels.
- Minimalista claro: white surface, slate text, blue/teal accents.
- Tech neon: near-black base, emerald/cyan focus, brighter glow.
- Elegante corporativo: deep navy, sky/emerald accents, restrained contrast.

## Spacing

- Page gutters: `16px` mobile, `24px` tablet, `32px` desktop.
- Section rhythm: `80px` to `96px` vertical padding.
- Panel padding: `16px` compact, `20px` default, `24px` feature.
- Control gap: `8px` to `16px`.

## Radius

- Cards and panels: `8px`.
- Inputs: `8px`.
- Inner preview items: `6px` to `8px`.
- Primary CTA buttons: pill radius for scan-friendly commands.

## Shadows

- Main panel: `0 24px 80px rgba(0, 0, 0, 0.2)`.
- Cyan focus glow: `0 0 32px rgba(34, 211, 238, 0.28)`.
- Preview elevation: `0 25px 50px rgba(0, 0, 0, 0.4)`.

## Button States

- Primary: cyan fill, slate text, glow on rest, brighter cyan on hover.
- Secondary: translucent white surface, subtle border, stronger border on hover.
- Ghost: transparent, muted text, low-opacity hover fill.
- Disabled: `opacity: 0.55` and pointer events disabled.

## Canva Asset Directions

- Cover image: dashboard plus generated landing hero in a dark premium frame.
- Promotional mockup: notebook and phone showing the same generated page.
- Social banner: pain-led headline with one CTA and a small preview crop.
- Thumbnail: compact before/after view of briefing to landing output.
